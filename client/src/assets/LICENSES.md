# Asset Attributions & Licenses

This project heavily relies on visual assets built by others, and it is critical they receive due credit.

To the best of my ability, I respect these properties' licenses:

- By default assets are not included in this repository.
- An asset will be included if, and only if, a license is not found or is permissive enough to allow distribution here.

This document aims to provide attribution and correct licensing to all content used, included or not.
